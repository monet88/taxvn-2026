## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Touch targets minimum 44x44px per Apple HIG.

---

## Visuals & Hierarchy

- **Focal Point:** The primary form (Auth inputs, calculation inputs) is the main visual anchor of the screen.
- **Hierarchy Order:** Display/Heading > Primary CTA Button > Primary Form Inputs > Secondary Actions > Tab Navigation.
- **Icon Actions:** Must include descriptive text labels or `accessibilityLabel` fallback for TalkBack/VoiceOver to ensure accessibility.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #ffffff | Background, main screen views |
| Secondary (30%) | #f9fafb | Cards, surfaces, sidebar, nav background |
| Accent (10%) | #059669 | Primary CTA, active tab icon, positive indicators |
| Destructive | #dc2626 | Error text, destructive actions, delete history |
| Primary Text | #1a1a1a | Standard text content |
| Border | #e5e7eb | Component borders |

Accent reserved for: Primary CTA button backgrounds (Đăng nhập tài khoản, Lưu lịch sử), active tab icon, positive/savings indicators. NEVER use for all interactive elements.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Đăng nhập tài khoản / Đăng ký tài khoản |
| Empty state heading | Chưa có lịch sử |
| Empty state body | Bạn chưa lưu kết quả nào. Hãy tính thử một loại thuế và bấm 'Lưu'. |
| Error state | Số không hợp lệ. Vui lòng nhập số tiền lớn hơn 0 và thử lại. |
| Destructive confirmation | Xóa lịch sử: Bạn có chắc chắn muốn xóa mục này khỏi lịch sử? |
