# Plan 02-03: History, Share, Push Token & Version Gate APIs

<objective>
Implement the remaining API routers: calculation history CRUD (save, list, filter, delete), share snapshot create/retrieve with NanoID tokens, push notification token registry, and tax-core version gate endpoint. All history/push routes require auth; share create is optional auth.
</objective>

<tasks>

<task id="03-01" title="History router (save, list, filter, delete)">
<read_first>
- apps/api/src/trpc.ts (protectedProcedure)
- apps/api/prisma/schema.prisma (CalculationHistory model)
- .planning/REQUIREMENTS.md (API-05, API-06, API-07)
</read_first>
<action>
Create `apps/api/src/routers/history.ts`:

**save**: protectedProcedure
- Input: `z.object({ toolName: z.string(), inputJson: z.record(z.unknown()), resultJson: z.record(z.unknown()), snapshotVersion: z.number().int(), taxCoreVersion: z.string() })`
- Create CalculationHistory record with ctx.user.userId
- Return `{ id, toolName, createdAt }`

**list**: protectedProcedure
- Input: `z.object({ toolName: z.string().optional(), dateFrom: z.string().datetime().optional(), dateTo: z.string().datetime().optional(), search: z.string().optional(), cursor: z.string().optional(), limit: z.number().int().min(1).max(50).default(20) })`
- Query with filters: toolName exact match, date range, search on toolName
- Order by createdAt DESC
- Cursor-based pagination using id
- Return `{ items: [...], nextCursor: string | null }`

**delete**: protectedProcedure
- Input: `z.object({ id: z.string() })`
- Delete where id AND userId match (prevent cross-user deletion)
- Return `{ success: true }`

**bulkDelete**: protectedProcedure
- Input: `z.object({ ids: z.array(z.string()).min(1).max(100) })`
- Delete where id IN ids AND userId matches
- Return `{ deletedCount: number }`

Wire into app router: `history: historyRouter`
</action>

<task id="03-02" title="Share router (create + retrieve)">
<read_first>
- apps/api/prisma/schema.prisma (ShareSnapshot model)
- .planning/phases/02-backend-api/02-CONTEXT.md (NanoID 8-char, 90-day expiration)
- .planning/REQUIREMENTS.md (API-10)
</read_first>
<action>
Create `apps/api/src/routers/share.ts`:

Import nanoid: `import { nanoid } from 'nanoid';`

**create**: publicProcedure (auth optional — check ctx.user)
- Input: `z.object({ snapshotJson: z.record(z.unknown()), snapshotVersion: z.number().int(), taxCoreVersion: z.string() })`
- Generate token: `nanoid(8)`
- Calculate expiresAt: new Date() + 90 days
- Create ShareSnapshot record
- If ctx.user exists, set createdById
- Return `{ token }`

**get**: publicProcedure
- Input: `z.object({ token: z.string().length(8) })`
- Find by token, check expiresAt > now
- If not found or expired → throw `NOT_FOUND` with message `SHARE_NOT_FOUND`
- Return `{ snapshotJson, snapshotVersion, taxCoreVersion, createdAt }`

Wire into app router: `share: shareRouter`
</action>
