<objective>
Scaffold a pnpm monorepo with Turborepo. Create three packages (tax-core, tax-data, config) and a root workspace configuration. The existing Next.js app stays as-is in the root for now — it will become apps/web in a future phase if needed.
</objective>

<tasks>

<task id="1.1">
<title>Create root workspace configuration</title>
<read_first>
- package.json
- tsconfig.json
</read_first>
<action>
1. Create `pnpm-workspace.yaml`:
```yaml
packages:
  - "packages/*"
  - "apps/*"
```

2. Create `.npmrc`:
```ini
auto-install-peers=true
shamefully-hoist=true
```

3. Update root `package.json`:
- Add `"workspaces"` field (for tooling compatibility): `["packages/*", "apps/*"]`
- Add scripts:
  - `"test": "turbo run test"`
  - `"build:packages": "turbo run build --filter='./packages/*'"`
  - `"lint:packages": "turbo run lint --filter='./packages/*'"`
- Keep existing scripts (`dev`, `build`, `start`, `lint`) for the Next.js app unchanged
- Change `"name"` to `"@taxvn/monorepo"`

4. Create `turbo.json`:
```json
{
  "$schema": "https://turbo.build/schema.json",
  "tasks": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**"]
    },
    "test": {
      "dependsOn": ["^build"],
      "outputs": []
    },
    "lint": {
      "dependsOn": ["^build"],
      "outputs": []
    },
    "dev": {
      "cache": false,
      "persistent": true
    }
  }
}
```
</action>
<acceptance_criteria>
- `pnpm-workspace.yaml` exists and contains `packages/*` and `apps/*`
- `.npmrc` exists and contains `auto-install-peers=true`
- Root `package.json` contains `"name": "@taxvn/monorepo"` and has `"test": "turbo run test"` script
- `turbo.json` exists and contains `"$schema": "https://turbo.build/schema.json"`
</acceptance_criteria>
</task>

</action>

- `packages/config/package.json` contains `"name": "@taxvn/config"`
- `packages/config/tsconfig.base.json` contains `"target": "ES2020"` and `"declaration": true`
</acceptance_criteria>
</task>

<task id="1.3">
<title>Create packages/tax-core scaffold</title>
<read_first>
- packages/config/tsconfig.base.json
</read_first>
<action>
1. Create `packages/tax-core/package.json`:
```json
{
  "name": "@taxvn/tax-core",
  "version": "1.0.0",
  "private": true,
  "main": "./dist/index.js",
  "types": "./dist/index.d.ts",
  "exports": {
    ".": {
      "types": "./dist/index.d.ts",
      "import": "./dist/index.js",
      "require": "./dist/index.js"
    }
  },
  "scripts": {
    "build": "tsc",
    "test": "vitest run",
    "lint": "tsc --noEmit"
  },
  "devDependencies": {
    "typescript": "^5.9.0",
    "vitest": "^3.1.1"
  }
}
```

2. Create `packages/tax-core/tsconfig.json`:
```json
{
  "extends": "../config/tsconfig.base.json",
  "compilerOptions": {
    "outDir": "./dist",
    "rootDir": "./src"
  },
  "include": ["src/**/*.ts"],
  "exclude": ["node_modules", "dist", "**/*.test.ts"]
}
```

3. Create `packages/tax-core/vitest.config.ts`:
```typescript
import { defineConfig } from 'vitest/config';

export default defineConfig({
  test: {
    include: ['src/**/*.test.ts'],
    globals: true,
  },
});
```

4. Create `packages/tax-core/src/index.ts`:
```typescript
// Placeholder — populated in Plan 02 (extraction)
export {};
```
</action>
<acceptance_criteria>
- `packages/tax-core/package.json` contains `"name": "@taxvn/tax-core"` and `"test": "vitest run"`
- `packages/tax-core/tsconfig.json` contains `"extends": "../config/tsconfig.base.json"`
- `packages/tax-core/vitest.config.ts` exists and contains `defineConfig`
- `packages/tax-core/src/index.ts` exists
</acceptance_criteria>
</task>
